# Contributing

## Rules

- Never commit source PDFs, extracted text, diagrams, screenshots, or derivative questions from copyrighted material.
- Keep resource records metadata-only: paths, hashes, descriptive classification, and rights status.
- Add tests for changed health/API behavior.
- Keep contracts in `packages/shared` versioned and backward-compatible where possible.

## Before pull request

- Run service checks from [local setup](docs/local-setup.md).
- Confirm secrets and `.env` files are absent.
- Confirm no prohibited content appears in staged files.
