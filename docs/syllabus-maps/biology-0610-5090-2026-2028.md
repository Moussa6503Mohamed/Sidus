## Authoring rules

- Every original question links to syllabus, map ID, objective ID, cognitive skill, and practical skill where relevant.
- Objective text requires rights/provenance review before storage. Store source location and a short internal interpretation, not copied syllabus text.
- Create no question until an editor verifies objective mapping and originality.
- Assess knowledge, data handling, and experimental skills. Do not treat topic completion as sufficient coverage.
